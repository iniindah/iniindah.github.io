P05-JQuery Sliding
Jquery-Efect Slide
Metode jQuery slideToggle() menampilkan atau menyembunyikan elemen yang dipilih dengan menganimasikan height mereka sedemikian rupa sehingga jika elemen tersebut awalnya ditampilkan, itu akan bergeser ke atas; jika disembunyikan, itu akan menjadi slide ke bawah yaitu beralih antara metode slideUp() dan slideDown() method slideUp() digunakan untuk menyembunyikan element (naik ke atas), sedangkan method slideDown() digunakan untuk memunculkan element (turun ke bawah), tentunya dengan efek sliding. Untuk mengontrol kecepatan efek, kita bisa menggunakan keyword “slow”, “fast”, atau angka sebagai argumen method ini (dalam satuan milisecond)..
